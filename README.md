### Requirements

1. [NodeJS](https://nodejs.org/)
2. [Meteor](https://www.meteor.com/)
3. [MongoDB](https://www.mongodb.com/)

### Installation

Clone the git repository:

```bash
$ git clone https://github.com/nirantak/pulse.git && cd pulse
```

Install necessary dependencies

```bash
$ meteor npm install
```

### Usage

Start the Server

```bash
$ meteor run
```
