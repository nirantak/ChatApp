### Installation

Clone the git repository:

```bash
$ git clone https://github.com/nirantak/pulse.git && cd pulse
```

Install necessary dependencies

```bash
$ meteor npm install
```

### Usage

Start the Server

```bash
$ meteor run
```

### Requirements

1. [MongoDB](https://www.mongodb.com/)
